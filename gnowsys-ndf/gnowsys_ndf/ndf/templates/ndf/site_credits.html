Welcome to the CLIx Platform


The CLIxPlatform is a Next Generation Digital Learning Platform powered by gstudio (a Node Description Framework (NDF) based knowledge representation software architecture). It integrates OpenAssessments for assessment questions and various other learning tools, available open source and/or developed as open source tools by CLIx.  The platform is a result of collaborative  work between technology teams at Tata Institute of Social Sciences, Mumbai, India; Homi Bhabha Centre for Science Education, TIFR, Mumbai, India; and, Massachussets Institute of Technology, Cambrige, USA.
