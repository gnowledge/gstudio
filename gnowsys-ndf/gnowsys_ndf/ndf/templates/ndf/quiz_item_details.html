About

Using Quiz Items, you can create a miscellaneous quiz-question. These quiz-questions can be of, basically three types: short response, single-choice and multiple-choice.

Using Quiz, you can build a quiz having list of quiz-items. Quiz-Items can be added to quiz in two ways:

  1. By editing, Quiz node.
  2. By using Quiz-Item button corresponding to the Quiz node (i.e. this will automatically add the quiz-item node to your quiz node).
